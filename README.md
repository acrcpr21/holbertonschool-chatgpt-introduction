# holbertonschool-chatgpt-introduction
## Description
This repository contains exercises and projects focused on using AI tools, specifically ChatGPT, to assist in software development and debugging. The goal of this project is to learn how to effectively prompt AI, understand its outputs, and apply the generated solutions to fix broken code across various scenarios.

## Contents

### Debugging directory (`/debugging`)
This folder contains various scripts and programs that initially contained bugs, which were identified and resolved using ChatGPT.

* **`change_background.html`**: A simple HTML/JavaScript page that dynamically changes the background color randomly when a button is clicked.
* **`checkbook.py`**: A Python class that simulates a simple checkbook. It allows a user to deposit, withdraw, and check their balance.
* **`factorial.py`**: A Python script that calculates the factorial of a given number using an iterative loop.
* **`factorial_recursive.py`**: A Python script that calculates the factorial of a given number using recursion.
* **`mines.py`**: A Python script implementation of the classic Minesweeper game.
* **`print_arguments.py`**: A Python script that iterates through and prints command-line arguments passed to it.
* **`tic.py`**: A Python script that runs an interactive, command-line Tic-Tac-Toe game for two players.

## Learning Objectives
* How to use ChatGPT to identify bugs in code.
* How to construct effective prompts to get the best technical answers.
* How to test and verify code provided by an AI.
* Understanding the limitations of AI-generated code.
